## 一、项目概述

| 项 | 说明 |
|---|------|
| **项目名称** | 寰引智能科技官网 |
| **公司全称** | 寰引智能科技（深圳）有限公司 |
| **网站定位** | 面向企业客户的 AI 转型落地服务商官网 |
| **核心目标** | 首页视觉冲击力拉满（科技感 + AI 感），让企业决策者一眼建立信任；其余页面承载 SEO + GEO |
| **目标受众** | 物流、跨境电商、报关、制造、制药、供应链等行业的企业老板/CTO/业务负责人 |
| **核心竞争力** | 10 家真实企业客户、11+ 落地系统、6 大行业解决方案 — 不是喊 AI 口号，是真正交付过 |

### 1.1 交付策略

本项目不采用优先级分级或分期开发。开发方式调整为**瀑布式一次性交付**：

1. PRD 中列出的页面、组件、SEO、GEO、响应式、可访问性、性能优化和测试全部纳入本次交付范围
2. AI 开发智能体按“搭建 → 内容数据 → 页面实现 → SEO/GEO → 表单/API → 测试验收”的顺序一次完成
3. 任何资料缺失时优先使用占位内容、环境变量或集中数据配置，不因缺少 Logo、域名、真实联系方式而中断开发
4. 交付完成标准不是“首页可看”，而是“全站可构建、可浏览、可测试、可部署”

## 二、技术架构

### 2.1 技术选型

| 层 | 选型 | 原因 |
|----|------|------|
| **框架** | Next.js 15 (App Router) | SSR/SSG 天然支持 SEO + GEO，首页可纯客户端渲染动画 |
| **样式** | Tailwind CSS 4 | CSS-first 配置（`@theme` 指令写入 `globals.css`），不再使用 `tailwind.config.ts` |
| **动画** | Framer Motion | React 生态最佳，页面滚动/Hero 进场动画 |
| **字体** | Google Fonts (Space Grotesk + DM Sans) | `next/font` 自动优化，零 CLS |
| **图标** | Lucide React | 轻量、统一、暗黑模式下线条清晰 |
| **部署** | Vercel | `vercel deploy` 一条命令，自动 HTTPS + CDN |
| **包管理** | npm | 系统已有 Node v25 + npm 11 |

## 三、设计系统规范

### 3.1 色彩系统

| Token | 色值 | 用途 | 对比度（vs 背景） |
|-------|------|------|-----------------|
| `--bg-base` | `#0F1117` | 页面主背景（深蓝灰，非纯黑） | — |
| `--bg-elevated` | `#1A1D27` | 卡片/区块背景（略亮于底色，做出层次） | — |
| `--bg-footer` | `#0A0C12` | 页脚背景（最暗） | — |
| `--text-primary` | `#F8FAFC` | 大标题、Hero 标题 | **17.2:1** AAA |
| `--text-body` | `#CBD5E1` | 正文段落、列表 | **11.6:1** AAA |
| `--text-muted` | `#94A3B8` | 标签、说明文字、次要信息（仅小字） | **6.4:1** AA+ |
| `--text-logo` | `#64748B` | Logo 墙文字、页脚次要文字（仅装饰） | — |
| `--text-copyright` | `#475569` | 版权信息（最低优先级） | — |
| `--accent` | `#6366F1` | 装饰元素、背景光晕、边框 hover（**不用于文字**） | — |
| `--accent-light` | `#C7D2FE` | 链接、胶囊标签文字（淡紫，仅小面积） | — |
| `--accent-glow` | `rgba(99,102,241,0.15)` | Hero 背景径向渐变光晕（仅装饰） | — |
| `--border-default` | `rgba(255,255,255,0.08)` | 卡片/分割线默认边框 | — |
| `--border-hover` | `rgba(99,102,241,0.4)` | 卡片 hover 时的强调边框 | — |
| `--cta-bg` | `#F8FAFC` | 主 CTA 按钮背景（白底黑字，最高对比） | **17.2:1** |
| `--cta-text` | `#0F172A` | 主 CTA 按钮文字 | — |

**配色铁律（人眼可读性优先）：**
1. 紫色 `#6366F1` **永远不直接用于正文文字**，仅用于背景光晕、装饰边框、图标背景圆
2. 发光效果 `box-shadow` **永远不碰文字**，仅用于卡片边框和装饰元素
3. 正文最低亮度 `#94A3B8`，不使用更暗的灰用于需要阅读的内容
4. 背景不用纯黑 `#000000`，改用深蓝灰 `#0F1117` 更柔和
5. CTA 按钮用白底黑字获得最强视觉引导

### 3.2 字体系统

| 字体 | 用途 | 字重 | CSS 变量 |
|------|------|------|---------|
| **Space Grotesk** | 所有标题 H1-H4、导航栏 Logo、数据指标数字 | 400/500/600/700 | `--font-heading` |
| **DM Sans** | 正文、列表、表单、按钮文字 | 400/500/700 | `--font-body` |

**字号阶梯：**

| Token | 大小 | 用途 |
|-------|------|------|
| `--text-hero` | 64px | Hero 主标题，font-weight: 700 |
| `--text-h1` | 48px | 区块大标题 |
| `--text-h2` | 32px | 区块小标题 |
| `--text-h3` | 24px | 卡片标题 / Hero 副标题 |
| `--text-body` | 18px | 正文 |
| `--text-small` | 14px | 标签/说明 |
| `--text-metric` | 56px | 数据指标数字，font-weight: 700 |

### 3.3 圆角 / 阴影 / 间距

**圆角：**
- `--radius-sm`: 8px（按钮、小卡片）
- `--radius-md`: 12px（标准卡片）
- `--radius-lg`: 16px（大卡片/容器）
- `--radius-full`: 9999px（胶囊形标签）

**阴影（仅卡片，绝不用于文字）：**
- `--shadow-card`: `0 1px 3px rgba(0,0,0,0.3)`
- `--shadow-hover`: `0 0 20px rgba(99,102,241,0.1), 0 1px 3px rgba(0,0,0,0.4)`

**间距系统（8px 基准）：**
```
4px → 8px → 12px → 16px → 24px → 32px → 48px → 64px → 96px → 128px
```

### 3.4 动效规范

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| Hero 标题进场 | 从下淡入 (translateY 20px → 0 + opacity 0→1) | 600ms | `ease-out` |
| 卡片 hover | 边框色变化 + 微妙阴影 | 200ms | `ease` |
| 数据指标 | 数字滚动 count-up（滚动到视口时触发） | 1500ms | `ease-out` |
| 行业 Tab 切换 | 内容区淡入淡出 | 300ms | `ease-in-out` |
| CTA 按钮 hover | 背景色微亮 + translateY(-1px) | 150ms | `ease` |
| 背景 | `prefers-reduced-motion: reduce` 时**全部禁用** | — | — |
